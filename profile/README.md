# Mangyomi

### The Modern, Extensible Manga Reader for Desktop

Mangyomi is a free, open-source manga reader application that brings your favorite manga, manhwa, and manhua sources together in one beautiful, uniform interface. Built with Electron and React, it offers a premium reading experience with powerful tracking and customization features.

## ‚ú® Key Features

-   **Extension System**: Access thousands of sources via a modular extension system.
-   **Tracking Integration**: Automatically sync your progress with **AniList**.
-   **Discord Rich Presence**: Show off what you're reading with automatic Discord status updates.
-   **Versatile Reading Modes**:
    -   *Vertical / Webtoon*: Smooth, continuous scrolling perfect for manhwa.
    -   *Horizontal*: Classic page-by-page reading for manga.
-   **Offline Reading**: Download chapters to read without an internet connection.
-   **NSFW Filtering**: Robust content filtering for your library and tags.
-   **Customizable UI**: Themes, zoom controls, and layout options to suit your preference.

## üì• Getting Started

### For Users
Download the latest release for Windows from the [Releases Page](https://github.com/Mangyomi/mangyomi-app/releases).

### For Developers
Mangyomi is built with the modern web stack:
-   **Electron**
-   **React**
-   **TypeScript**

To build from source:

```bash
git clone https://github.com/Mangyomi/mangyomi-app.git
cd mangyomi-app
npm install
npm run dev
```
